**Schema — three tables. CRITICAL: Do NOT use `GENERATED ALWAYS AS` computed columns** (Pitfall 1 in RESEARCH.md — fails on Python 3.9's bundled SQLite). Compute `failure_rate` in SELECT queries instead:

```sql
CREATE TABLE IF NOT EXISTS analysis_runs (
    run_id              TEXT PRIMARY KEY,
    build_id            INTEGER NOT NULL,
    created_at          TEXT NOT NULL,
    total_tests         INTEGER DEFAULT 0,
    failed_tests        INTEGER DEFAULT 0,
    total_input_tokens  INTEGER DEFAULT 0,
    total_output_tokens INTEGER DEFAULT 0,
    estimated_cost_usd  REAL DEFAULT 0.0,
    report_json         TEXT
);

CREATE TABLE IF NOT EXISTS test_results (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    run_id          TEXT NOT NULL,
    test_title      TEXT NOT NULL,
    full_title      TEXT NOT NULL,
    status          TEXT NOT NULL,
    category        TEXT,
    probability     REAL DEFAULT 0.0,
    retry_count     INTEGER DEFAULT 0,
    passed_on_retry INTEGER DEFAULT 0,
    FOREIGN KEY (run_id) REFERENCES analysis_runs(run_id)
);

CREATE TABLE IF NOT EXISTS flakiness_index (
    test_full_title TEXT PRIMARY KEY,
    total_runs      INTEGER DEFAULT 0,
    failure_count   INTEGER DEFAULT 0,
    flaky_count     INTEGER DEFAULT 0,
    last_updated    TEXT NOT NULL
);
```

**`async def init_db(db_path: Path = DB_PATH) -> None`:**
- Open connection with `aiosqlite.connect(db_path)`
- Use `executescript()` to run the three CREATE TABLE IF NOT EXISTS statements in one call
- Call `await db.commit()` after executescript
- Safe to call on every startup (idempotent)

**`async def store_result(run_id: str, result, db_path: Path = DB_PATH) -> None`:**
- `result` is a `ClassificationResult` instance
- INSERT into test_results: run_id, test_title, full_title, status="unexpected" (all results coming in are failing/flaky), category, probability, retry_count (from result if available — add `retry_count: int = 0` to ClassificationResult if not already there), passed_on_retry=1 if result.category in {"race_condition"} and result.method != "rule_engine" else 0
- Actually: passed_on_retry should reflect whether test.status was "flaky" — the caller passes this. Add parameter `passed_on_retry: bool = False` to store_result.

**`async def upsert_flakiness(test_full_title: str, failed: bool, flaky: bool, db_path: Path = DB_PATH) -> None`:**
- READ current row first (INSERT OR REPLACE resets AUTOINCREMENT — use UPDATE if exists, INSERT if not)
- Use pattern from RESEARCH.md Pattern 5: read current (total, failures, flakies), then INSERT OR REPLACE with incremented values
- Set last_updated to `datetime('now')` in SQL or `datetime.utcnow().isoformat()` in Python

**`async def store_run_cost(run_id: str, build_id: int, total_input_tokens: int, total_output_tokens: int, estimated_cost_usd: float, total_tests: int, failed_tests: int, report_json: str = "", db_path: Path = DB_PATH) -> None`:**
- INSERT OR REPLACE into analysis_runs with all fields
- `created_at` = `datetime.utcnow().isoformat()`

**`async def get_run_cost(run_id: str, db_path: Path = DB_PATH) -> dict | None`:**
- SELECT all columns from analysis_runs WHERE run_id = ?
- Set `db.row_factory = aiosqlite.Row` before query
- Return `dict(row)` or None if not found
- Note: dict(aiosqlite.Row) is required for JSON serializability (Pitfall 7 in RESEARCH.md)

**`async def get_flakiness_index(db_path=DB_PATH, min_runs=MIN_RUNS_FOR_FLAKINESS) -> list[dict]`:**
- Query flakiness_index WHERE total_runs >= min_runs
- Compute failure_rate in SQL: `CAST(failure_count AS REAL) / total_runs AS failure_rate`
- ORDER BY failure_rate DESC
- Set db.row_factory = aiosqlite.Row; return `[dict(r) for r in rows]`
- Import DB_PATH from .db

**`def compute_endpoint_heatmap(correlated_tests: list) -> list[dict]`:**
- `correlated_tests` is a `list[CorrelatedTest]`
- Use `defaultdict(int)` for endpoint_totals and endpoint_failures
- For each CorrelatedTest, for each EndpointMatch in ct.endpoint_matches: increment `endpoint_totals[ep.kg_id]` and `endpoint_failures[ep.kg_id]`
- Return list of dicts sorted by failure_rate descending: `[{"endpoint": kg_id, "total_failing_tests": total, "failure_rate": failures/total}]`
- Pure Python — no SQLite, no I/O, no pandas

**`def compute_failure_ratios(classifications: list[dict]) -> dict`:**
- `classifications` is a list of dicts, each with a `"category"` key (string)
- Count into four buckets: infra, app, test_design, uncertain
- Return: `{"counts": {"infra": N, "app": N, "test_design": N, "uncertain": N}, "ratios": {...}, "total": N}`
- Handle total=0 by returning 0.0 ratios (not ZeroDivisionError)

**src/classifier/report.py** — report generator and token accumulator:

**`RunTokenAccumulator` dataclass:**
```python
from dataclasses import dataclass, field

@dataclass
class RunTokenAccumulator:
    total_input_tokens: int = 0
    total_output_tokens: int = 0
    haiku_calls: int = 0
    sonnet_calls: int = 0

    def add(self, result) -> None:
        """Add token usage from a ClassificationResult."""
        if result.method == "llm_haiku":
            self.haiku_calls += 1
        elif result.method == "llm_sonnet":
            self.sonnet_calls += 1
        self.total_input_tokens += result.tokens_used  # note: tokens_used is total (input+output) for simplicity
        # If we need split tracking, ClassificationResult.tokens_used stores combined for now

    @property
    def total_tokens(self) -> int:
        return self.total_input_tokens

    def estimated_cost_usd(self) -> float:
        """Rough post-run cost estimate using haiku/sonnet split."""
        from .cost_estimator import HAIKU_INPUT_COST_PER_M, SONNET_INPUT_COST_PER_M
        # Simplified: treat all tokens as input (conservative overestimate)
        haiku_fraction = self.haiku_calls / max(self.haiku_calls + self.sonnet_calls, 1)
        avg_cost_per_m = haiku_fraction * HAIKU_INPUT_COST_PER_M + (1 - haiku_fraction) * SONNET_INPUT_COST_PER_M
        return round((self.total_input_tokens / 1_000_000) * avg_cost_per_m, 4)
```

**`def generate_report(classifications: list, run_id: str, accumulator: RunTokenAccumulator) -> dict`:**
- `classifications` is a `list[ClassificationResult]`
- Returns a dict with the full structured report (OUT-01 format):

```python
{
    "run_id": run_id,
    "summary": {
        "total_classified": len(classifications),
        "by_category": {...},    # count per category
        "by_method": {...},      # count per method (rule_engine/llm_haiku/llm_sonnet)
        "token_usage": {
            "total_tokens": accumulator.total_tokens,
            "haiku_calls": accumulator.haiku_calls,
            "sonnet_calls": accumulator.sonnet_calls,
            "estimated_cost_usd": accumulator.estimated_cost_usd(),
        },
    },
    "tests": [
        {
            "test_title": r.test_title,
            "full_title": r.full_title,
            "category": r.category,
            "probability": r.probability,
            "method": r.method,
            "fix_recommendation": r.fix_recommendation,
            "summary_paragraph": r.summary_paragraph,
            "reasoning_chain": r.reasoning_chain,
        }
        for r in classifications
    ],
}
```
